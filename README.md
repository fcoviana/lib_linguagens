# Lib linguagens - Estudo npm


#### Dependências
- NodeJs
- NPM

------------

#### Configurar o usuário NPM
###### E-mail
```shell
    npm set init.author.email “email@mail.com”
```
###### Name
```shell
    npm set init.author.name “User Name”
```
###### Licença
```shell
    npm set init.license “MIT”
```

------------

#### Comandos lib
###### Baixar a lib
```shell
    npm i lib_linguagens
```
